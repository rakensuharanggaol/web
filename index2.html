Tugas 1 - Menampilkan data dalam javascript

Berikut adalah 3 cara menampilkan data dalam javascript:

  1. Menggunakan document.write()

    2. Metode ini akan menuliskan langsung ke dalam halaman web.

    
    <script>
      document.write("<p>Ini adalah data yang ditampilkan dengan document.write()</p>");
    </script>
  2. Menggunakan innerHTML

    3. Metode ini akan memanipulasi konten HTML dari elemen yang sudah ada.

    
    <script>
      var paragraf = document.getElementById("paragraf");
      paragraf.innerHTML = "<p>Ini adalah data yang ditampilkan dengan innerHTML</p>";
    </script>
    <p id="paragraf"></p>
  3. Menggunakan createElement()

    4. Metode ini akan membuat elemen baru yang kemudian ditambahkan ke dalam halaman web.

    
    <script>
      var paragraf = document.createElement("p");
      paragraf.textContent = "Ini adalah data yang ditampilkan dengan createElement()";
      document.body.appendChild(paragraf);
    </script>

Tugas 2 - Implementasi variabel, operator, struktur kondisi dan perulangan

Berikut adalah contoh halaman HTML dengan implementasi variabel, operator, struktur kondisi dan perulangan.


<!DOCTYPE html>
<html>
<head>
  <title>Perpustakaan</title>
</head>
<body>

<h2>Daftar Buku</h2>

 <script>
  var buku = [
    {judul: "Buku A", harga: 50000},
    {judul: "Buku B", harga: 70000},
    {judul: "Buku C", harga: 30000}
  ];

  var totalHarga = 0;

  for (var i = 0; i < buku.length; i++) {
    document.write("<p>" + buku[i].judul + " - Rp " + buku[i].harga + "</p>");
    totalHarga += buku[i].harga;
  }

  document.write("<p>Total Harga: Rp " + totalHarga + "</p>");

  if (totalHarga > 100000) {
    document.write("<p>Total harga melebihi Rp 100000</p>");
  } else {
    document.write("<p>Total harga kurang dari Rp 100000</p>");
  }
</script>

</body>
</html>
